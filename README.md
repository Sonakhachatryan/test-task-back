## Message Sender
This app is demonstration of sending emails.

## INSTALLATION
  - clone the project
  - create `.env` file
  - copy content from `env.example` to `.env`
  - configure `env` according to your settings
  - run following commands
    ```shell
    $ composer install
    $ php artisan migrate
    $ npm install
    $ npm run production
    ```
    
## RUNNING APPLICATION    
  - run following command
    ```shell
    $ php artisan serve
    ```
